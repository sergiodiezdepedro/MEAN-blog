MongoDB

MongoDB es una base de datos libre, de código abierto, multiplataforma y basada en documentos. Esta base de datos no-SQL utiliza documentos del tipo JSON con esquemas dinámicos.

ExpressJS

Express es una infraestructura de aplicaciones web Node.js mínima y flexible que proporciona un conjunto sólido de características para las aplicaciones web y móviles.

Angular 2

Angular es un framework de JavaScript de código abierto, desarrollado por Google, que se utiliza para crear y mantener aplicaciones web de una sola página.

NodeJS

Entorno en tiempo de ejecución multiplataforma, de código abierto, para el servidor, asíncrono, con I/O de datos en una arquitectura orientada a eventos y basado en el motor V8 de Google.

JavaScript

JavaScript es un lenguaje de programación interpretado, base del stack MEAN. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.

Blog

Un blog o bitácora es un sitio web que incluye, a modo de diario personal de su autor o autores, contenidos de su interés, actualizados con frecuencia y a menudo comentados por los lectores.​
